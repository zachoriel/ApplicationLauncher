# AppLauncher

A lightweight C# WPF form for managing software applications which deal with regular updates. 

**COMPLETED FEATURES:**

-File downloading

-Automatic update checking

-Version tracking

-Download progress bar

-Multi-app selection

-User-initiated downloads (rather than automatic)
